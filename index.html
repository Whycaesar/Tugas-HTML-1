HTML, CSS & JavaScript

HTML

Hyper Text Markup Languange (HTML) adalah sebuah bahasa yang digunakan untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah web internet dan pemformatan hiperteks sederhana yang ditulis dalam berkas format ASCII agar dapat menampilkan tampilan wujud yang terintegrasi. Dalam kata lain, berkas yang dibuat dalam perangkat lunak pengolahan kata dan disimpan dalam format ASCII normal sehingga menjadi halaman web dengan perintah-perintah HTML. Bermula dari sebuah bahasa yang sebelumnya banyak digunakan didunia penerbitan. HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web. HTML saat ini merupakan standar internet yang didefinisikan dan dikendalikan penggunannya oleh World Wide Web Consortium (W3C). HTML dibuat oleh kolaborasi Caillau TIM & Berners-Lee Robert ketika mereka bekerja di CERN (lembaga penelitian fisika energi tinggi di Jenewa) pada tahun 1989.

Sejarah

Berawal pada tahun 1980 seorang ahli fisika, Tim Berners-Lee dan juga seorang kontraktor CERN (Organisasi Eropa untuk Riset Nuklir) mengusulkan dan menyusun ENQUIRE, sebuah sistem untuk para ilmuan dalam membagi dokumen. Sembilan tahun kemudian, Berners-Lee mempublikasikan HTML dan menulis jaringan beserta perangkat lunaknya di akhir 1990. Pada tahun yang sama, Berners-Lee & Robert Cailliau, insinyur sistem data CERN berkolaborasi dalam sebuah permintaan untuk pendanaan, namun tidak diterima secara resmi oleh CERN. Penjelasan pertama yang dibagi untuk umum "Tanda HTML", pertama kali disebutkan di Internet oleh Tim Berners-Lee pada akhir tahun 1991. Tanda ini menggambarkan 18 elemen awal mula, versi sederhana dari HTML. Kecuali untuk tag Hyperlink.

    Markah/Tanda

    Secara garis besar, terdapat 4 jenis elemen di HTML:

  • Struktural. Tanda yang menentukan level atau tingkatan dari sebuah tulisan.
  • Presentasional. Tanda yang menentukan tampilan dari sebuah tulisan tidak perduli dengan level tulisan tersebut.
  • Hiperteks. Tanda yang menunjukkan pranala ke bagian dari dokumen tersebut atau pranalan ke dokumen lain.
  • Widget. Elemen yang membuat objek-objek lain seperti tombol(button), list dan garis horizontal.

    • CSS

    Cascading Style Sheet (CSS) merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemograman.
    Sama halnya styles dalam aplikasi pengolahan kata seperti Microsoft Word yang dapat mengatur beberapa style, misalnya heading, subbab, bodytext, footer, images dan style lainnya untuk dapat digunakan bersama-sama dalam beberapa berkas(file). Pada umumnya CSS digunakan untuk memformat tampilan halaman web yang dibuat dengan bahasa HTML.

    Sejarah CSS

    Nama CSS didapat dari fakta bahwa setiap deklarasi style yang berbeda dapat diletakkan secara berurutan, yang kemudian membentuk ayah dan anak (parent-child) pada setiap style. CSS sendiri merupakan sebuah teknologi internet yang direkomendasikan oleh World Wide Web Consortium (W3C) pada tahun 1996.

    Versi

    Untuk saat ini terdapat 3 versi CSS, yaitu CSS1, CSS2 dan CSS3. CSS1 dikembangkan berpusat pada pemformatan HTML, CSS2 dikembangkan untuk memenuhi kebutuhan terhadap format dokumen agar bisa ditampilkan di printer. Sedangkan CSS3 adalah versi terbaru dari CSS yang mampu melakukan banyak hal dalam desain website. CSS2 mendukung penentuan posisi konten, downloadable, huruf font, tampilan pada table/table layout dan media tipe untuk printer. Kehadiran versi CSS yang ketiga diharapkan lebih baik dari versi yang pertama dan kedua.
    CSS3 juga dapat melakukan animasi pada halaman website, diantaranya animasi warna hingga animasi 3D. Dengan CSS3 desainer lebih dimudahkan dalam hal kompabilitas websitenya pada smartphone dengan menggunakan fitur baru yakni media query. selain itu banyak fitur baru pada CSS3 seperti: multiple background, border-radius, drop-shadow, border image, CSS Math dan CSS Object Model.

    Sifat CSS

    Ada dua sifat CSS yakni internal dan eksternal. Jika internal yang dipilih, maka skrip itu dimasukkan secara langsung ke halaman website yang akan didesain. Kalau halaman web yang lain akan didesain dengn model yang sama, maka skrip CSS itu harus dimasukkan lagi ke dalam halaman web yang lain itu. Sedangkan sifat yang kedua adalah eksternal dimana skrip CSS dipisahkan dan diletakkan dalam berkas khusus, sehingga dibutuhkan semacam tautan menuju berkas CSS itu jika halaman web yang didesain akan dibuat seperti model yang ada di skrip tersebut.

    JavaScript

    Adalah bahasa pemrograman tingkat tinggi dan dinamis, Javascript populer di internet dan dapat bekerja di sebagian besar penjelajah web populer seperti Internet Explorer(IE), Mozilla, Firefox, Netscpae dan opera.

    Sejarah

    JavaScript pertama dikembangkan oleh Brendan Eich dari Netscape dibawah nama Mocha, yang nantinya namanya diganti menjadi LiveScript dan akhirnya menjadi JavaScript.
    Navigator sebelumnya telah mendukung Java untuk lebih bisa dimanfaatkan para programmer yang non-java. Maka dikembangkanlah bahasa program Live-Script untuk mengakomodasi hal tersebut. Bahasa inilah yang akhirnya berkembang menjadi JavaScript, walaupun tidak ada hubungan antara Java dan JavaScript.
